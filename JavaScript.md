   > ### 用规定的文法，来表达特定的语义，最终操作运行时

    + #### 词法
      + 直接量、关键字、运算符
    + #### 语法
      + 表达式、语句、函数、对象、模块，

  + ### 运行时

    + #### 类型（数据结构）
      + 类型（对象、数字、字符串）
      + 实例（应用和机制）
    + #### 执行过程（算法）
      + 事件循环
      + 函数执行
      + 微任务执行
      + 语句的执行
